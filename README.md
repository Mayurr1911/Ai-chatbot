# AI Chatbot

A simple yet powerful AI chatbot built using HTML, CSS, and JavaScript. This chatbot integrates with Google's Gemini AI to provide comprehensive, intelligent responses.

## Features

- **Interactive UI**: Clean and user-friendly chat interface
- **AI-Powered Responses**: Integrated with Google Gemini 2.5 Flash API for intelligent conversations
- **Real-time Messaging**: Send messages via button click or Enter key
- **Responsive Design**: Works on various screen sizes
- **Fast Responses**: Get detailed answers within 10 seconds

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Google Gemini AI API

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ai-chatbot.git
   cd ai-chatbot
   ```

2. Open `index.html` in your web browser.

## Usage

1. Open `index.html` in a web browser.
2. Type your message in the input field.
3. Click the "Send" button or press Enter to send your message.
4. The AI will respond with comprehensive information, links, and suggestions.

## API Setup

The chatbot uses Google's Gemini AI API. The API key is already configured in the `script.js` file.

**Note**: For security reasons, consider moving the API key to a server-side environment variable in production.
